Welcome!

This website contains the undergraduate thesis project of Sierra Allred.

This project focuses on helping future college students decide what institution they would like to attend based on cost analysis.

There are 3 phases to this project: Visualization, Database, and Statistical Analysis.

Visualization
This phase of the project was done as part of a team. Documentation is found under “Phase 1: Visualization”, as well as a tutorial on how to use the visualization. The visualization itself can be found under “College Analyzer”.

Database
This phase of the project focuses on connecting a database to the visualization, and was done individually. Documentation is found under “Phase 2: Database”

Statistical Analysis
This phase of the project focuses on predicting student debt, and was done individually. Documentation is found under “Phase 3: Statistical Analysis”
